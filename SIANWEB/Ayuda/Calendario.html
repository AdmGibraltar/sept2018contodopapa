Calendario 


En la parte superior derecha de la pantalla se despliega el icono de acceso al modulo de Calendario.
Si el usuario tiene actividades pendientes se le enumeran dentro de un recuadro rojo, de manera que sea facilmente identificable.



Al darle doble click sobre ese icono, se despliega la ventana del Calendario de Actividades asignadas al usuario.
Para los RSC, las actividades definidas son:

  • Visita Personal [En azul]
  • Llamada Telefonica [En verde]
  • Correo Electronico [En naranja]



La vista mensual es la que se despliega por default pero es posible cambiarla para ver unicamente la semana en curso.


Y en el caso de que sea necesario, se pueden visualizar las actividades de un solo dia.


Se puede revisar a detalle cada una de las actividades, al darle doble click sobre ellas, de manera que aparezca una pantalla con sus datos completos.


Al hacerlo sobre una actividad de Correo Electronico tenemos acceso a un formato institucional para enviarlo directamente al cliente sin necesidad de entrar a nuestro correo.
Una copia de dicho correo se enviara a nuestro buzon para seguimiento, de la misma manera, el cliente al contestar el correo lo hara a nuestra direccion sin problema.


Para el caso de las llamadas telefonicas, la informacion desplegada es muy similar.


Dentro de esta pantalla de detalle tenemos acceso a una guia para la llamada telefonica, asi como a una encuesta de calidad del servicio para llenarla cuando se hable con el cliente.


Para el caso de las visitas personales, se despliega tambien la informacion referente a los prerequisitos (en caso de haberse definido asi).


Cuando sea necesario, es posible mover la cita para una nueva fecha, con solo arrastrarla a la nueva fecha
Debe tenerse en consideracion que existen dias inhabiles, en los cuales no es posible agendar ningun tipo de actividad.


El calendario pedira un motivo para dicho movimiento, el cual de no proporcionarse cancela la nueva fecha.


Si por algun motivo se requiere eliminar una cita, bastara con darle click en el icono de la esquina superior derecha de cada una de ellas.


El modulo solicita un motivo para dicha eliminacion, nuevamente, de no proporcionarlo, la accion se cancelara y la cita permanece activa en el calendario.


Es posible filtrar las citas desplegadas para ver solamente las de un RSC o las de un cliente en especifico.


El calendario mantendra este filtro aun cuando se cambie el rango de tiempo que se esta revisando.
